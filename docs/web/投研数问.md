# 投研数问前端页面说明

## 1. 导航入口

- 在 `web` 顶部导航新增页签：`投研数问`。
- 不影响既有页签：`股票列表`、`投资助手`。

## 2. 页面布局

页面采用左右分栏，整体参考豆包聊天交互。

- 左栏（会话区）：
  - 上部分：
    - `新建会话` 按钮
    - `可查询范围` 总结卡片（来自 query-scope 接口）
  - 下部分：
    - `会话历史` 列表
    - 每条会话支持选中与删除
- 右栏（聊天区）：
  - 聊天气泡区（用户问题 + 投研数问回复）
  - 输入框（Enter 发送，Shift+Enter 换行）
  - 发送按钮

## 3. 对接接口（SQL Copilot）

基于 `docs/api/sql-copilot-api.md` 与 `docs/spec/sql-copilot.spec.md`：

- 获取可查询范围：
  - `GET /api/v1/sql-copilot/query-scope`
  - 页面展示 `scope_summary`
- 获取会话历史列表：
  - `POST /api/v1/sql-copilot/sessions/list`
- 删除会话：
  - `POST /api/v1/sql-copilot/sessions/delete`
- 发起问答：
  - `POST /api/v1/sql-copilot/chat`
  - 页面展示字段：
    - `answer`（中文总结）
    - `sql`（默认折叠展示）
    - `error`（错误态展示）

## 4. 交互说明

- 新建会话：
  - 前端进入“空会话输入态”，首次发送问题时由后端返回 `session_id`。
- 选择历史会话：
  - 切换右侧聊天内容（当前版本按本地会话内缓存展示）。
- 删除会话：
  - 二次确认后删除，并刷新会话列表。
- 错误处理：
  - 接口失败显示错误文案；
  - 业务失败优先展示 `error` / `answer`。
